# 🐝 TaskBee - Task Management System

📌 Overview
TaskBee is a Java-based task management application designed to help users organize, manage, and track their tasks efficiently. The project focuses on applying Object-Oriented Programming (OOP) principles to build a structured and scalable system.

🎯 Project Objective
To develop a simple yet effective task management system that improves productivity by allowing users to create, manage, and organize tasks through a user-friendly interface.

🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Basic Software Development Life Cycle (SDLC) practices

✨ Key Features
- Task creation and management
- Goals creation and management
- Organized task structure using OOP principles
- Simple and user-friendly interface

👥 Team Collaboration
This project was developed as part of a collaborative team effort, where all members contributed to planning, development, and design following basic SDLC methodology.
